# Liam's Game — Concept Art Bundle

This bundle contains the concept art created during the Chapter 1 art pass.

## Recommended repo destination

Copy these folders into your repo like this:

```text
assets/
  concept/
    key-art/
    environments/
    characters/
```

## Suggested next step

1. Drop these files into `assets/concept/`
2. Keep all versions for reference
3. Later, copy your selected “keeper” images into production folders like:
   - `assets/maps/`
   - `assets/portraits/characters/`
   - `assets/icons/`
