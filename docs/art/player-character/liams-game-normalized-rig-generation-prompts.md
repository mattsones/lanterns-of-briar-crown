# Liam's Game — Exact Generation Prompt Set for Final Normalized Rig Templates

These are the prompt texts to use as the baseline reference for the approved normalized rig set.

## Shared style block

```text
Storybook fantasy character art with a hand-drawn, hand-painted feel, subtle painterly texture, soft brushwork, gentle ink-like definition, and a warm illustrated-book atmosphere. Cozy fantasy tone, sincere and adventurous, family-friendly, with gentle whimsy. Practical rustic clothing, transparent background, no scenery.
```

## Shared technical block

```text
Full-body character on a transparent background, centered on the canvas. Use a slight 3/4 front-facing body orientation with the head turned to look directly at the viewer. Relaxed but ready stance. The character's right hand is the main hand and appears on the viewer-left side of the image. The character's left hand is the off hand and appears on the viewer-right side of the image. Keep feet planted on a shared baseline, preserve clear headroom for helmets and hats, preserve readable chest and waist zones, and keep the overall rig suitable for paper-doll equipment layering.
```

## Shared negative block

```text
No photorealism, no 3D look, no anime, no glossy game-render style, no background scene, no props, no weapons, no armor, no hat, no cloak, no text, no labels, no extra figures.
```
